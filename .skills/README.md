# Skills

Reusable AI-assisted workflows for valkey-search development.

## Usage

Skills in this directory can be invoked by AI coding assistants (Claude Code, GitHub Copilot, etc.) to perform common development tasks.

## Adding a New Skill

Create a markdown file in this directory with:

1. A descriptive title as an H1 heading
2. A brief description of what the skill does
3. A `## Steps` section with the procedure to follow

## Available Skills

| Skill | Description |
|-------|-------------|
| [stale-prs.md](stale-prs.md) | Find open PRs that haven't been updated in 30+ days |
